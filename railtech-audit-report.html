3. Доступность и UX

ПараметрСтатусРекомендация
Mobile-friendly Отлично Viewport мета-тег присутствует. При 375px мобильная версия корректно адаптируется: бургер-меню, вертикальная раскладка, читабельный текст.
Alt-тексты изображений Критично 12 из 22 изображений без alt-текста (55%). Среди них — логотип компании. Каждому изображению нужен описательный alt. Для декоративных — alt="" + role="presentation".
Формы без label Критично 6 полей ввода без label / aria-label. Скринридеры не смогут озвучить назначение этих полей. Добавьте aria-label или <label for="...">.
Landmark <main> Критично Отсутствует элемент <main>. Скринридеры используют его для навигации к основному контенту. Оберните основной контент в <main>.
Skip-to-content ссылка Нужно исправить Отсутствует skip-link. Добавьте скрытую ссылку «Перейти к содержимому» для клавиатурной навигации.
ARIA-атрибуты Отлично 17 элементов с ARIA-ролями. Tabindex не содержит нестандартных значений.

Контрастность текста Отлично Светлый текст (#e2e8f0) на тёмном фоне (#0f172a) — коэффициент контраста ~15:1, значительно выше требуемого WCAG AA (4.5:1).

Резюме: ТОП-5 приоритетных шагов

  1. Добавить alt-тексты к 12 изображениям без описания
    Критично для SEO и доступности. Логотип компании должен иметь alt="RailTech". Декоративные элементы — alt="". Время: ~30 минут.
  2. Добавить <main>, skip-link и label для форм
    Обернуть контент в <main>, добавить aria-label к 6 полям ввода, создать skip-to-content ссылку. Время: ~1 час.
  3. Добавить defer ко всем <script> тегам
    Все 4 JS-файла блокируют рендеринг. Добавление defer ускорит First Contentful Paint. Также добавьте preload для критических ресурсов. Время: ~15 минут.
  4. Расширить Schema.org (JSON-LD)
    Добавить разметку WebSite (с SearchAction), LocalBusiness (адрес в Ташкенте, телефон, email), BreadcrumbList. Это повысит качество сниппетов в поиске. Время: ~1 час.
  5. Перевести мультиязычность на подпапки вместо query-параметров
    Заменить ?lang=en / ?lang=uz на /en/ / /uz/. Поисковики лучше индексируют URL-пути. Обновить sitemap.xml и hreflang-теги. Время: ~3–4 часа.
